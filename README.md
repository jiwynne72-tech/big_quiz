# 🎮 JW BrainBox Academy

> A growing collection of educational games, quizzes and interactive learning tools by **James Wynne**.

![JW BrainBox](icon_JW.png)

## 🌟 About

**JW BrainBox Academy** is a collection of browser-based educational games designed to make learning enjoyable.

All games are written in pure **HTML, CSS and JavaScript**, meaning there is **nothing to install** and they work directly in any modern web browser.

The collection covers:

- 🌍 Geography
- 🏳️ World Flags
- 🧠 Mental Mathematics
- 🎭 Shakespeare
- 👑 British History
- 🏛️ Latin
- 🇬🇷 Ancient Greek
- 🎲 English Vocabulary
- 🧩 Logic & Brain Teasers

The project is continually expanding with new educational titles.
